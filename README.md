# StaySync — Hotel & Guesthouse Management Website

StaySync is a modern, premium full-stack hotel and guesthouse management system built using the MERN stack.

## Architecture
- **Frontend**: React, Vite, Tailwind CSS v3, and shadcn/ui.
- **Backend**: Node.js, Express, MongoDB (Mongoose).
- **Notifications**: Telegram Bot API integration.
- **Reports**: Dynamic PDF invoice generation.

## Local Setup

### Backend Setup
1. Navigate to the backend directory:
   ```bash
   cd hotel-booking-backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up your `.env` variables (refer to `.env.example`).
4. Run the seed script:
   ```bash
   npm run seed:demo
   ```
5. Start the dev server:
   ```bash
   npm run dev
   ```

### Frontend Setup
1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the dev server:
   ```bash
   npm run dev
   ```
